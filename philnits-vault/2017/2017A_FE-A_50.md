# 2017A_FE-A_50

According to the ISO/IEC 12207 (Systems and software engineering - Software life cycle processes (SLCP)), the Software Implementation Process has the following lower-level processes: 1) Construction Process 2) Requirements Analysis Process 3) Architectural and Detailed Design Process 4) Integration and Testing process. Which of the following is an appropriate order for the Software Implementation Process?

a) $1)\rightarrow2)\rightarrow3)\rightarrow4)$
b) $2)\rightarrow3)\rightarrow1)\rightarrow4)$
c) $2)\rightarrow3)\rightarrow4)\rightarrow1)$
d) $3)\rightarrow4)\rightarrow2)\rightarrow1)$
?
b) $2)\rightarrow3)\rightarrow1)\rightarrow4)$
### Explanation

The standard software development lifecycle order follows:

1. **Requirements Analysis** → understand what to build
2. **Architectural and Detailed Design** → plan how to build it
3. **Construction** → write the code
4. **Integration and Testing** → verify the components work together

This maps to: **2 → 3 → 1 → 4**
