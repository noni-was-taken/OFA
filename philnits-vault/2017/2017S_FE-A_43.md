# 2017S_FE-A_43
A data backup system must support restoring all transactions until 5 minutes before a disaster happens. What type of goal is this?

a) Maximum tolerable downtime
b) Recovery point objective
c) Recovery time objective
d) Service level agreement
?
b) Recovery point objective
### Explanation

**RPO (Recovery Point Objective)** defines the **maximum acceptable amount of data loss** measured in time — i.e., how far back the system can be restored to. "Restoring all transactions until 5 minutes before a disaster" means the RPO is 5 minutes.

- **RTO (Recovery Time Objective)**: how quickly the system must be restored after a disaster
- **Maximum tolerable downtime**: the longest a system can be offline before causing unacceptable harm
- **SLA**: a broader service agreement
