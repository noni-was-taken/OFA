# 2017S_FE-A_20
The methods for representing the results of syntactic analysis in a compiler include quadruplet format: (Operator, Operand 1, Operand 2, Result). For which of the following expressions does the series of quadruplets below represent the syntactic analysis?

`(*, B, C, T1), (/, T1, D, T2), (+, A, T2, T3)`

a) $A+B*C/D$
b) $A+B*C/T2$
c) $B*C+A/D$
d) $B*C+TI/D$
?
a) $A+B*C/D$
### Explanation

Read the quadruplets sequentially:
1. `(*, B, C, T1)` → $T1 = B \times C$
2. `(/, T1, D, T2)` → $T2 = (B \times C) / D$
3. `(+, A, T2, T3)` → $T3 = A + (B \times C / D)$

This corresponds to $\mathbf{A + B * C / D}$, respecting standard operator precedence.
