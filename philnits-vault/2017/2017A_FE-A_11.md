# 2017A_FE-A_11

Which of the following is a memory device that stores programs or data in factories, and only has the read operation for the stored programs or data?

a) DRAM
b) Flash memory
c) Mask ROM
d) SRAM
?
c) Mask ROM
### Explanation

A **Mask ROM** is programmed during the **manufacturing process** at the factory. Once manufactured, the contents are **permanent and read-only** — no write or erase operations are possible.

The other options support read/write operations:
- **DRAM** → volatile, read/write RAM (main memory)
- **Flash memory** → non-volatile but electrically erasable and writable
- **SRAM** → volatile, fast read/write RAM (cache memory)
