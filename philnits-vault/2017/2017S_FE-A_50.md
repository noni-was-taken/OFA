# 2017S_FE-A_50
When a software component is tested, which of the following is an appropriate combinations of X and Y? X: It calls the software component to be tested. Y: It is called from the software component to be tested.

a) X=component, Y=unit
b) X=driver, Y=stub
c) X=stub, Y=driver
d) X=unit, Y=component
?
b) X=driver, Y=stub
### Explanation

In unit/component testing:

- **Test Driver**: code that **calls** (drives) the component under test, simulating the calling module → **X** ✅
- **Test Stub**: code that **replaces** a module **called by** the component under test → **Y** ✅
