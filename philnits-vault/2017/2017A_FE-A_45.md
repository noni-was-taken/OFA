# 2017A_FE-A_45

Which of the following is the most appropriate description of polymorphism in object oriented design?

a) It allows developers to define classes incrementally by reusing classes defined previously as the basis for new classes.
b) It allows the analyst to focus on the important dimensions while ignoring nonessential dimensions.
c) It combines the processes and data into a single object.
d) It is the provision of a single interface to entities of different types.
?
d) It is the provision of a single interface to entities of different types.
### Explanation

**Polymorphism** allows a single interface (e.g., a method name) to operate on different types or behave differently depending on the object type — "many forms" from one interface.

The other options describe different OOP concepts:
- **A** → inheritance
- **B** → abstraction
- **C** → encapsulation
