# 2017S_FE-A_31
Which of the following SQL statements for the table 'ShipmentRecord' obtains the largest value?

| ProductNumber | Date       | Quantity |
|---------------|------------|----------|
| NP200         | 2015-10-10 | 3        |
| FP233         | 2015-10-10 | 2        |
| NP200         | 2015-10-11 | 1        |
| FP233         | 2015-10-11 | 2        |

a) SELECT AVG(Quantity) FROM ShipmentRecord WHERE ProductNumber = 'NP200'
b) SELECT COUNT(*) FROM ShipmentRecord
c) SELECT MAX(Quantity) FROM ShipmentRecord
d) SELECT SUM (Quantity) FROM ShipmentRecord WHERE Date = '2015-10-11'
?
b) SELECT COUNT(*) FROM ShipmentRecord
### Explanation

Calculate each result:
- **a)** `AVG` for NP200: $(3+1)/2 = 2$
- **b)** `COUNT(*)`: counts all rows = **4** ✅ (largest)
- **c)** `MAX(Quantity)`: = 3
- **d)** `SUM` on 2015-10-11: $1+2 = 3$

Option **b** returns **4**, the largest value.
