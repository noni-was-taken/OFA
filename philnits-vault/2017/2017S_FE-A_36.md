# 2017S_FE-A_36
Which of the following is an appropriate description of the netstat utility?

a) It displays active TCP connections.
b) It displays the contents of the DNS resolver cache.
c) It displays the mapping between IP addresses and corresponding MAC addresses in the cache.
d) It displays the route path to the specified destination and measures the packet delays.
?
a) It displays active TCP connections.
### Explanation

**`netstat`** displays network connection information, including **active TCP/UDP connections**, listening ports, routing tables, and interface statistics.

- Option b) describes `ipconfig /displaydns` (DNS cache)
- Option c) describes the **`arp -a`** command (ARP cache)
- Option d) describes **`traceroute`** / **`tracert`**
