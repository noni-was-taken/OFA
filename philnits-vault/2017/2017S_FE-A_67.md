# 2017S_FE-A_67
Which of the following is an appropriate description of an RFI?

a) It is a document describing the basic policy for the system, including a summary of the system, the purpose, the necessary functions, the system requirements, and contract matters.
b) It is a document that is created after an RFP.
c) It is a document used by companies looking for implementation of computerization to request specific system proposals from prospective suppliers such as system vendors.
d) It is a document used to request information concerning computerization from prospective suppliers such as system vendors.
?
d) It is a document used to request information concerning computerization from prospective suppliers such as system vendors.
### Explanation

**RFI (Request for Information)** is sent to potential vendors to **gather general information** about their capabilities before deciding whether to issue an RFP.

- Option a) describes a **system requirements specification**
- Option b) is incorrect — an RFI typically comes *before* an RFP
- Option c) describes an **RFP (Request for Proposal)**
