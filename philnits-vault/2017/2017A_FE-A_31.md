# 2017A_FE-A_31

When a PC connected to a LAN is starting up, which of the following protocols is used to automatically assign an IP address to the PC?

a) DHCP
b) DNS
c) FTP
d) PPP
?
a) DHCP
### Explanation

**DHCP (Dynamic Host Configuration Protocol)** automatically assigns IP addresses, subnet masks, default gateways, and DNS server addresses to devices when they join a network — including at startup.

The other protocols serve different purposes:
- **DNS** → resolves domain names to IP addresses
- **FTP** → file transfer protocol
- **PPP** → Point-to-Point Protocol for serial links
