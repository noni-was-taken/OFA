# 2017A_FE-A_27

Which of the following is an appropriate description of the mapping between the relational model and relational database as its implementation?

a) A domain is mapped to a character type or a character string type.
b) A relation is mapped to a table.
c) Attributes and columns are ordered from left to right.
d) Neither tuples nor rows have duplicates.
?
b) A relation is mapped to a table.
### Explanation

The standard mappings between the relational model and its implementation are:

| Relational Model | Database Implementation |
|-----------------|------------------------|
| **Relation** | **Table** |
| Tuple | Row |
| Attribute | Column |
| Domain | Set of allowable values |

The other options are incorrect:
- **A** → a domain maps to **any** data type, not only character types
- **C** → in the relational model, attributes are **unordered**
- **D** → tuples have no duplicates in the model, but SQL tables **can** have duplicate rows unless a constraint is enforced
