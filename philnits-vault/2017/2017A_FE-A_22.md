# 2017A_FE-A_22

Which of the following is a logic gate combination that provides an output 1 when both inputs are 0?

a) AND or XOR
b) NAND or XOR
c) NOR or XNOR
d) OR or XNOR
?
c) NOR or XNOR
### Explanation

Testing each gate with inputs A=0, B=0:

| Gate | Output |
|------|--------|
| AND | 0 AND 0 = **0** |
| NAND | NOT(0 AND 0) = **1** |
| OR | 0 OR 0 = **0** |
| **NOR** | NOT(0 OR 0) = **1** ✓ |
| XOR | 0 XOR 0 = **0** |
| **XNOR** | NOT(0 XOR 0) = **1** ✓ |

Both **NOR** and **XNOR** output 1 when both inputs are 0.
