# 2017S_FE-A_16
Consider a RAID 0 (i.e., block-level striping without parity or mirroring) system with two disks; when data is distributed evenly across the two disks and both disks have a failure rate of 0.1%, what is the overall failure rate (as a percentage) of this RAID system?

a) 0.0001
b) 0.0100
c) 0.1999
d) 1.9990
?
c) 0.1999
### Explanation

In **RAID 0**, the system fails if **either** disk fails.

Let failure rate per disk = $p = 0.001$.

$$P(\text{success}) = (1 - 0.001)^2 = 0.999^2 = 0.998001$$

$$P(\text{failure}) = 1 - 0.998001 = 0.001999 = \mathbf{0.1999\%}$$
