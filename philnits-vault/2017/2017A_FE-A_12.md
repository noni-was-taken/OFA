# 2017A_FE-A_12

Which of the following is an appropriate role of a device driver?

a) It decides the next task to be executed from tasks queued up for execution.
b) It directly controls the hardware according to requests from an application program.
c) It interprets the command strings that are entered by the user, and starts the applicable program.
d) It manages the display status on the screen in multiple windows.
?
b) It directly controls the hardware according to requests from an application program.
### Explanation

A **device driver** is software that acts as an interface between the OS/applications and a hardware device. It translates high-level commands into the device-specific operations needed to control the hardware directly.

The other options describe different system components:
- **A** → CPU scheduler
- **C** → command-line shell / interpreter
- **D** → window manager
