# 2017S_FE-A_19
In virtual memory management, which of the following page replacement methods replaces the page for which the most time has passed since it was last used?

a) FIFO
b) LFU
c) LIFO
d) LRU
?
d) LRU
### Explanation

**LRU (Least Recently Used)** replaces the page that has **not been accessed for the longest time**. It is based on the principle of temporal locality.

- **FIFO**: replaces the page that has been in memory the longest (first in = first out), regardless of usage
- **LFU (Least Frequently Used)**: replaces the page accessed the fewest times
- **LIFO**: replaces the most recently loaded page
