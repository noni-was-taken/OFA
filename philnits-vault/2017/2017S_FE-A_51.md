# 2017S_FE-A_51
According to the Capability Maturity Model Integration, which is the final stage and has the highest level of maturity?

a) Auditable
b) Custom
c) Optimized
d) Quantitative
?
c) Optimized
### Explanation

The **CMMI** defines five maturity levels:

| Level | Name |
|-------|------|
| 1 | Initial |
| 2 | Managed |
| 3 | Defined |
| 4 | Quantitatively Managed |
| **5** | **Optimizing** ✅ |

At Level 5, the organization focuses on **continuous process improvement** using quantitative feedback. This is the highest and final maturity level.
