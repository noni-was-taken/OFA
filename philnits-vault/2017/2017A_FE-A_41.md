# 2017A_FE-A_41

Which of the following is an appropriate terminology concerning the behavior that documents retrieved from distinct origins are isolated from each other?

a) Clickjacking
b) Cross-Origin Resource Sharing (CORS)
c) Cross-Site Request Forgery (CSRF)
d) Same-Origin Policy
?
d) Same-Origin Policy
### Explanation

The **Same-Origin Policy** is a browser security rule that **isolates** documents from different origins — scripts on one origin cannot access resources (e.g., DOM, cookies) from another origin.

The other options describe related but different concepts:
- **CORS** → a mechanism that **allows** cross-origin requests as an exception to the Same-Origin Policy
- **Clickjacking** → a UI redress attack tricking users into clicking hidden elements
- **CSRF** → an attack that tricks a user's browser into making unintended requests to another site
