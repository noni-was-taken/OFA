# 2017S_FE-A_2
Which of the following is logically equivalent to the expression below? Here A, B, C, and D are integers, '<' is a less-than operator, and '<=' is a less-than-or-equal-to operator.

$\text{not } ((A \le B) \text{ or } (C < D))$

a) $(A < B)$ and $(C \le D)$
b) $(A < B)$ or $(C \le D)$
c) $(B < A)$ and $(D \le C)$
d) $(B < A)$ or $(D \le C)$
?
c) $(B < A)$ and $(D \le C)$
### Explanation

Apply **De Morgan's Law**: `NOT(P OR Q)` = `NOT P AND NOT Q`

- `NOT(A <= B)` → `B < A`
- `NOT(C < D)` → `D <= C`

Combined with AND: $(B < A)$ **and** $(D \le C)$
