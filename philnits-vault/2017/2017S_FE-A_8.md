# 2017S_FE-A_8
For a two-dimensional integer array a, whose element $a(i,j)$ is of value $2i+j$, what is the value $a(a(1,1) \times 2, a(2,2)+1)$?

a) 12
b) 13
c) 18
d) 19
?
d) 19
### Explanation

Using the formula $a(i,j) = 2i + j$:

- $a(1,1) = 2(1)+1 = 3$
- $a(2,2) = 2(2)+2 = 6$

Compute outer indices:
- Row: $3 \times 2 = 6$
- Column: $6 + 1 = 7$

Final: $a(6,7) = 2(6)+7 = 12+7 = \mathbf{19}$
