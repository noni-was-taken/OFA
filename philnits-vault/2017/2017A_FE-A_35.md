# 2017A_FE-A_35

Which of the following is a form of malware that requires a host in order to propagate from one computer to another?

a) Trojan horses
b) Viruses
c) Worms
d) Zombies (Bots)
?
b) Viruses
### Explanation

A **virus** attaches itself to a legitimate **host file** or program and spreads when that host file is executed or shared with other systems — it **requires a host** to propagate.

The other options propagate or operate differently:
- **Worm** → self-replicating, spreads **independently** over networks without needing a host file
- **Trojan horse** → disguises itself as legitimate software but does not self-replicate
- **Bot/Zombie** → a compromised computer controlled remotely; not primarily defined by its propagation method
