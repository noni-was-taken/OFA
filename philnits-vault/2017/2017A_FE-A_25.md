# 2017A_FE-A_25

In a DBMS, which of the following functions decides the schema?

a) Definition function
b) Maintenance function
c) Recovery function
d) Security function
?
a) Definition function
### Explanation

The **definition function** of a DBMS is responsible for defining the **schema** — the structure of the database including tables, data types, constraints, and relationships — using DDL (Data Definition Language) commands like CREATE, ALTER, and DROP.

The other functions serve different roles:
- **Maintenance function** → handles data insertion, update, deletion (DML)
- **Recovery function** → restores the database after a failure
- **Security function** → manages access control and authentication
