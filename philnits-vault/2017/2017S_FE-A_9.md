# 2017S_FE-A_9
A recursive function f(n) is defined as follows for any natural number n. What is the value f(5)?

$f(n) = 1 \text{ (if } n \le 1\text{)}$

$f(n) = n + f(n-1) \text{ (otherwise)}$

a) 6
b) 9
c) 15
d) 25
?
c) 15
### Explanation

Unroll the recursion:

- $f(5) = 5 + f(4)$
- $f(4) = 4 + f(3)$
- $f(3) = 3 + f(2)$
- $f(2) = 2 + f(1) = 2 + 1 = 3$

Working back up: $f(3)=6$, $f(4)=10$, $f(5) = 5+10 = \mathbf{15}$

This is simply $\sum_{k=1}^{5} k = \frac{5 \times 6}{2} = 15$.
