# 2017A_FE-A_49

Which of the following is a method for analyzing a source and object codes, and extract program specifications and design information?

a) Reengineering
b) Refactoring
c) Restructuring
d) Reverse engineering
?
d) Reverse engineering
### Explanation

**Reverse engineering** goes from existing code **back to design and specification information** — analyzing source or object code to understand how a system was built. Generating a UML diagram from code is a classic example.

The other options describe different processes:
- **Reengineering** → redesigning or rebuilding a system; may include reverse engineering as a step
- **Refactoring** → restructuring code **without changing external behavior** to improve readability
- **Restructuring** → reorganizing code structure (a subset of reengineering)
