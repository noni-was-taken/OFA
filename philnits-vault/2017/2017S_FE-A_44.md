# 2017S_FE-A_44
Which of the following is an appropriate test to verify the efficacy of security measures by attempting an actual intrusion?

a) Exception test
b) Functional test
c) Penetration test
d) Stress test
?
c) Penetration test
### Explanation

A **penetration test (pen test)** is a simulated cyberattack performed by authorized security professionals to **actively attempt to exploit vulnerabilities**, verifying whether security controls can withstand real-world attacks.

- **Exception test**: tests how a system handles invalid inputs or error conditions
- **Functional test**: verifies the system performs its intended functions correctly
- **Stress test**: evaluates system behavior under extreme load conditions
