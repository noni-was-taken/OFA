# 2017A_FE-A_15

Which of the following is an appropriate characteristic of a web system implemented in a three-layer client/server system configuration?

a) A business process is executed on the server side, and the client software displays the result on a screen according to the HTML coding.
b) A program coded in HTML is run on the server side, and the client software displays the result on a screen.
c) Every time a business process is changed, a business process application for operating the web system must be sent and installed on the client terminal.
d) On the client terminal, it is necessary to have a resident service program waiting to receive an HTTP request from the server side.
?
a) A business process is executed on the server side, and the client software displays the result on a screen according to the HTML coding.
### Explanation

In a **three-layer client/server architecture**, the layers are:
- **Presentation (client)** → browser renders HTML
- **Application/Business Logic (app server)** → executes business logic
- **Data (database server)** → stores data

Option A correctly describes this: business logic runs **server-side**, and the client (browser) **renders HTML results**.

The other options are incorrect:
- **B** → HTML is a markup language, not a server-side programming language
- **C** → one of the key merits of web systems is that updates are deployed server-side without client reinstallation
- **D** → the client initiates HTTP requests to the server, not the reverse
