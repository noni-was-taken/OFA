# 2017A_FE-A_30

Which of the following is located at layer 3 of the OSI basic reference model, and performs path selection and packet relay for an end-to-end communication?

a) Data link layer
b) Network layer
c) Session layer
d) Transport layer
?
b) Network layer
### Explanation

| Layer | Number | Function |
|-------|--------|----------|
| Physical | 1 | Bit transmission over a medium |
| Data link | 2 | Frame delivery within a single network segment |
| **Network** | **3** | **Routing / path selection across networks using IP** |
| Transport | 4 | End-to-end reliability (TCP/UDP) |
| Session | 5 | Session management |

IP operates at Layer 3 — the **Network layer** — which performs path selection and packet relay.
