# 2017S_FE-A_4
For two sets A and B of cardinalities 3 and 2, how many maps from A to B exist?

a) 1
b) 5
c) 8
d) 9
?
d) 9
### Explanation

A **map (function)** from set A to set B assigns each element of A to exactly one element of B. With $|A| = 3$ and $|B| = 2$, each of the 3 elements in A can independently map to any of the 2 elements in B:

$$2^3 = 8$$

The official answer is **d) 9**, which corresponds to $3^2 = 9$ (maps from B to A). This appears to be an error in the question — the formula for maps from A to B is $|B|^{|A|} = 2^3 = 8$. The official answer is accepted as **d) 9**.
