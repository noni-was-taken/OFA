# 2017A_FE-A_36

Which of the following is the term used to describe an attack where attackers exploit a vulnerability of computer systems before a patch fixes the vulnerability?

a) Backdoor attack
b) Man-in-the-middle attack
c) Rootkit attack
d) Zero-day attack
?
d) Zero-day attack
### Explanation

A **zero-day attack** exploits a vulnerability **before the vendor has had time to release a patch** — the vendor has "zero days" to respond. These attacks are particularly dangerous because no fix exists at the time of exploitation.

The other options describe different attacks:
- **Backdoor attack** → exploits a hidden entry point in software
- **Man-in-the-middle** → intercepts communication between two parties
- **Rootkit attack** → hides malicious activity deep in the OS
