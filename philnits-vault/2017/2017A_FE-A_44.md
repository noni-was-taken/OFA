# 2017A_FE-A_44

When a packet filtering firewall controls packets by using a list of rules, which of the following is the rule that is applied to packet A and its action? Here, the firewall applies the rules shown in the list below in the order number 1-4, and if a rule is applied, then the remaining rules are not applied. [List of rules: 1 | 10.1.2.3 | * | * | * | * | Blocked; 2 | * | 10.2.3.* | TCP | * | 25 | Permitted; 3 | * | 10.1.* | TCP | * | 25 | Permitted; 4 | * | * | * | * | * | Blocked] [Packet A: Source 10.1.2.3, Dest 10.2.3.4, Protocol TCP, Source Port 2100, Dest Port 25]

a) Blocked by number 1
b) Blocked by number 4
c) Permitted by number 2
d) Permitted by number 3
?
a) Blocked by number 1
### Explanation

The firewall checks rules **in order, top to bottom**, and stops at the first match.

**Rule 1**: Blocks any packet with source IP **10.1.2.3**

Packet A's source IP is **10.1.2.3** → Rule 1 matches immediately → **Blocked by Rule 1**

The remaining rules are never checked.
