# 2017A_FE-A_6

When Bubble sort algorithm is used, how many exchange operations are required to sort the following numbers in ascending order? 9, 2, 13, 21, 3, 0

a) 7
b) 8
c) 9
d) 10
?
c) 9
### Explanation

Simulating bubble sort pass by pass:

**Pass 1:** 9↔2, 21↔3, 21↔0 → 3 swaps → `2, 9, 13, 3, 0, 21`

**Pass 2:** 13↔3, 13↔0 → 2 swaps → `2, 9, 3, 0, 13, 21`

**Pass 3:** 9↔3, 9↔0 → 2 swaps → `2, 3, 0, 9, 13, 21`

**Pass 4:** 3↔0 → 1 swap → `2, 0, 3, 9, 13, 21`

**Pass 5:** 2↔0 → 1 swap → `0, 2, 3, 9, 13, 21` ✓

**Total: 3 + 2 + 2 + 1 + 1 = 9 swaps**
