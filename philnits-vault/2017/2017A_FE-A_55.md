# 2017A_FE-A_55

What is the function point value of a program that has the functions and characteristics described in the table below? Here, the correction coefficient of complexity is 0.75. [Table: External input count 1, weight 4; External output count 2, weight 5; Internal logical file count 1, weight 10; External interface file count 0, weight 7; External inquiry count 0, weight 4]

a) 18
b) 24
c) 30
d) 32
?
a) 18
### Explanation

**Step 1: Calculate unadjusted function points**

$$UFP = (1 \times 4) + (2 \times 5) + (1 \times 10) + (0 \times 7) + (0 \times 4)$$
$$UFP = 4 + 10 + 10 + 0 + 0 = 24$$

**Step 2: Apply the correction coefficient**

$$FP = UFP \times 0.75 = 24 \times 0.75 = 18$$
