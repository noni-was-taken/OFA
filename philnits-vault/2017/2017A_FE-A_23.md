# 2017A_FE-A_23

Which of the following uses a flip-flop circuit in a memory cell?

a) DRAM
b) EEPROM
c) SDRAM
d) SRAM
?
d) SRAM
### Explanation

**SRAM (Static RAM)** uses a **flip-flop circuit** (a bistable latch made of transistors) for each bit. This makes it fast and requires no refresh, but uses more transistors per bit than DRAM.

The other options use different storage mechanisms:
- **DRAM / SDRAM** → uses a **capacitor** to store each bit (requires periodic refresh)
- **EEPROM** → uses floating-gate transistors (non-volatile, electrically erasable)
