# 2017S_FE-A_34
In an IPv4 network, which of the following is the correct combination of an IP address class, its IP address range, and a subnet mask?

a) Class A: 1.0.0.0–127.255.255.255, Mask: 255.0.0.0
b) Class B: 128.0.0.0–191.255.255.255, Mask: 255.255.0.0
c) Class C: 192.168.0.0–223.255.255.255, Mask: 255.255.255.0
d) Class D: 224.0.0.0-239.255.255.255, Mask: 255.255.255.255
?
b) Class B: 128.0.0.0–191.255.255.255, Mask: 255.255.0.0
### Explanation

| Class | Range | Default Subnet Mask |
|-------|-------|---------------------|
| A | 0.0.0.0 – 127.255.255.255 | 255.0.0.0 |
| **B** | **128.0.0.0 – 191.255.255.255** | **255.255.0.0** ✅ |
| C | 192.0.0.0 – 223.255.255.255 | 255.255.255.0 |
| D (Multicast) | 224.0.0.0 – 239.255.255.255 | N/A |

- Option a) starts at 1.0.0.0 but Class A starts at 0.0.0.0
- Option c) incorrectly starts Class C at 192.168.0.0 instead of 192.0.0.0
- Option d) Class D has no standard subnet mask
