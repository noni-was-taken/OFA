# 2017S_FE-A_27
Which of the following is a database design that consists of multiple tables, with rows and columns that are linked together through matching data stored in each table?

a) Hierarchical database
b) Network database
c) Object-oriented database
d) Relational database
?
d) Relational database
### Explanation

A **relational database** organizes data into **tables (relations)** of rows and columns. Tables are linked through **common key values** (primary/foreign keys), allowing data to be queried and joined across tables using SQL.

- **Hierarchical**: data organized in a tree structure (parent-child)
- **Network**: data organized as a graph with multiple parent-child links
- **Object-oriented**: data stored as objects with attributes and methods
