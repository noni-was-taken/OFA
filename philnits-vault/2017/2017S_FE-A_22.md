# 2017S_FE-A_22
Which of the following is a method that performs integration of the modules that configure a program during execution of the program?

a) Dynamic linking
b) Interpreter
c) Overlay
d) Static linking
?
a) Dynamic linking
### Explanation

**Dynamic linking** defers the binding of library modules until **runtime**. The required modules (e.g., DLL files on Windows, .so files on Linux) are loaded and linked into memory only when needed during execution.

- **Static linking**: all modules are combined into a single executable at **compile/link time**
- **Interpreter**: executes source code line by line at runtime; a different concept
- **Overlay**: a memory management technique for fitting large programs into limited RAM
