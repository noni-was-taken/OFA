# 2017S_FE-A_64
Among the 'four categories' of the enterprise architecture, which of the following completes the list after business architecture, technology architecture, and application architecture?

a) Bus architecture
b) Data architecture
c) Software architecture
d) System architecture
?
b) Data architecture
### Explanation

The **four domains of Enterprise Architecture** are:
1. **Business Architecture**: business strategy, governance, processes
2. **Data Architecture**: structure of an organization's data assets ✅
3. **Application Architecture**: individual applications and interactions
4. **Technology Architecture**: IT infrastructure, hardware, networks

**Data architecture** is the missing fourth element.
