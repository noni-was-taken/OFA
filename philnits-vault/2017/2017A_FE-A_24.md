# 2017A_FE-A_24

Which of the following is used to control the visual styles of HTML documents, such as text sizes, text colors, or page layout?

a) CMS
b) CSS
c) RSS
d) Wiki
?
b) CSS
### Explanation

**CSS (Cascading Style Sheets)** is the stylesheet language that controls the **visual presentation** of HTML documents — defining fonts, colors, spacing, layout, and more.

The other options are unrelated to visual styling:
- **CMS** (Content Management System) → platform for managing web content (e.g., WordPress)
- **RSS** (Really Simple Syndication) → a web feed format for content distribution
- **Wiki** → a collaborative web platform for editing content
