# 2017A_FE-A_28

When a storage location is to be calculated from a key value, which of the following methods enables the same calculation result to be obtained from different key values?

a) B+ tree index
b) Bitmap index
c) Hash index
d) Inverted index
?
c) Hash index
### Explanation

A **hash index** uses a hash function to map key values to storage locations. Because multiple different keys can produce the same hash value, this is called a **collision** — the same calculation result from different key values.

B+ tree, bitmap, and inverted indexes do not compute storage locations via a hash function and do not have this collision property.
