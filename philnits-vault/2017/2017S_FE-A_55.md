# 2017S_FE-A_55
According to EVM (Earned Value Management), the budget of a software development project is $10,000 and the project duration is planned for 10 weeks. After 4 weeks, it is revealed that only 30% of the work has been completed against a cost of $4,200. What is the cost performance index?

a) 0.126
b) 0.300
c) 0.420
d) 0.714
?
d) 0.714
### Explanation

**EVM formulas:**
- **EV (Earned Value)**: $10,000 \times 30\% = \$3,000$
- **AC (Actual Cost)**: $\$4,200$
- **CPI**: $\frac{EV}{AC} = \frac{3000}{4200} \approx \mathbf{0.714}$

A CPI below 1.0 indicates the project is **over budget** — spending more than the value of work produced.
