# 2017S_FE-A_11
Which of the following is the computer architecture where processors execute multiple instruction streams on multiple data streams in parallel?

a) MIMD
b) MISD
c) SIMD
d) SISD
?
a) MIMD
### Explanation

Flynn's Taxonomy:

| Type | Instructions | Data | Example |
|------|-------------|------|---------|
| **SISD** | Single | Single | Classic single-core CPU |
| **SIMD** | Single | Multiple | GPU, vector processors |
| **MISD** | Multiple | Single | Fault-tolerant systems |
| **MIMD** | Multiple | Multiple | Multi-core CPUs, clusters |

**MIMD** — Multiple Instruction, Multiple Data — matches the description.
