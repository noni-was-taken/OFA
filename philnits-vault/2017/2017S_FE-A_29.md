# 2017S_FE-A_29
Which of the following is an appropriate normalization form (NF) used in a relational database in which every non-prime attribute must be dependent only on the primary key, and no non-prime attribute is dependent on another non-prime attribute?

a) 1NF
b) 2NF
c) 3NF
d) BCNF
?
c) 3NF
### Explanation

The description matches **Third Normal Form (3NF)**:
- The table is in 2NF (no partial dependencies)
- **No transitive dependencies**: no non-prime attribute depends on another non-prime attribute

| Normal Form | Requirement |
|-------------|-------------|
| 1NF | All values are atomic |
| 2NF | No partial dependencies on the primary key |
| **3NF** | No transitive dependencies between non-key attributes ✅ |
| BCNF | Every determinant is a candidate key |
