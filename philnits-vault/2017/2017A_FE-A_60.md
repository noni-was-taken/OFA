# 2017A_FE-A_60

In an information security audit, which of the following is a check item for confirming availability?

a) Error checks during data input to be performed appropriately.
b) Interruption duration to be managed so that the standard of the SLA specifying the interruption duration can be satisfied
c) The database to be encrypted
d) The unauthorized removal of an external memory medium to be prohibited
?
b) Interruption duration to be managed so that the standard of the SLA specifying the interruption duration can be satisfied
### Explanation

The three pillars of information security (CIA) map to each option:

| Option | Security Property |
|--------|------------------|
| A — Input error checks | Integrity |
| **B — Managing downtime / SLA compliance** | **Availability** |
| C — Database encryption | Confidentiality |
| D — Preventing unauthorized removal | Confidentiality |

**Availability** means the system is accessible when needed. Managing **interruption duration** against an SLA directly measures and ensures availability.
