# 2017S_FE-A_48
As shown in the list below, there are three types of coupling. Which of the following is the most appropriate combination of the type of coupling? Coupling A: One module passes an element of control to the other module. Coupling B: One module passes the data structure as a parameter to the other module. Coupling C: Two modules have access to the same global data.

a) Coupling A=Content coupling, Coupling B=Data coupling, Coupling C=Common coupling
b) Coupling A=Content coupling, Coupling B=Stamp coupling, Coupling C=External coupling
c) Coupling A=Control coupling, Coupling B=Data coupling, Coupling C=External coupling
d) Coupling A=Control coupling, Coupling B=Stamp coupling, Coupling C=Common coupling
?
d) Coupling A=Control coupling, Coupling B=Stamp coupling, Coupling C=Common coupling
### Explanation

Module coupling types:

| Coupling Type | Description |
|---------------|-------------|
| **Control** | One module passes a control flag to direct another module's logic ✅ (A) |
| **Stamp** | A data structure (record/object) is passed as a parameter ✅ (B) |
| **Common** | Two modules share the same global data ✅ (C) |
| Content | One module directly modifies another's internal data |
| External | Modules share an externally imposed data format |
| Data | Only simple data values (not structures) are passed |
