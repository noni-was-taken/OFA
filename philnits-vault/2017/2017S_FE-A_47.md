# 2017S_FE-A_47
Which of the following are components of a UML2.x (including 2.0 and later versions) state diagram?

a) Message and Activity
b) Message and Transition
c) State and Message
d) State and Transition
?
d) State and Transition
### Explanation

A **UML State Diagram** models the behavior of a system or object through its **states** and **transitions** between them triggered by events.

- **State**: a condition or situation in the life of an object
- **Transition**: a change from one state to another

**Messages** belong to **Sequence Diagrams**; **Activities** belong to **Activity Diagrams**.
