# 2017S_FE-A_32
Which of the following is a routable IP address in the Internet?

a) 10.129.205.3
b) 172.40.77.4
c) 192.168.10.7
d) 192.168.203.5
?
b) 172.40.77.4
### Explanation

**Private (non-routable) IP ranges** (RFC 1918):
- `10.0.0.0` – `10.255.255.255`
- `172.16.0.0` – `172.31.255.255`
- `192.168.0.0` – `192.168.255.255`

Checking each:
- a) `10.129.205.3` → private
- **b) `172.40.77.4` → public ✅** (172.40 is outside the 172.16–172.31 range)
- c) `192.168.10.7` → private
- d) `192.168.203.5` → private
