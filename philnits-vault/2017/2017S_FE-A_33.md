# 2017S_FE-A_33
Among the seven layers of the OSI basic reference model, which layer is responsible for forwarding packets, including routing them through intermediate routers?

a) The data link layer
b) The network layer
c) The session layer
d) The transport layer
?
b) The network layer
### Explanation

The **Network Layer (Layer 3)** handles **logical addressing** (IP addresses) and **routing** — determining the best path for packets across multiple networks through intermediate routers.

- **Data link layer (Layer 2)**: frame delivery between directly connected nodes (MAC addresses)
- **Transport layer (Layer 4)**: end-to-end communication, flow control, error recovery
- **Session layer (Layer 5)**: manages sessions/connections between applications
