# 2017A_FE-A_52

Which of the following is an appropriate description of 'Lead' relationship in project time management?

a) It is the amount of time by which a successor activity can be advanced with respect to a predecessor activity.
b) It is the amount of time by which a successor activity will be delayed with respect to a predecessor activity.
c) It is the amount of time that a schedule activity can be delayed or extended from its early start date without delaying the project completion date or violating a schedule constraint.
d) It is the number of work periods (hours, days) required to complete individual activities with the estimated resources.
?
a) It is the amount of time by which a successor activity can be advanced with respect to a predecessor activity.
### Explanation

In project scheduling:
- **Lead** → allows a successor activity to **start earlier** (overlap) relative to the predecessor — essentially a negative lag
- **Lag** → a **delay** applied to a successor (option B)
- **Float/Slack** → time a task can be delayed without affecting the project end date (option C)
- **Duration** → total time to complete an activity (option D)
