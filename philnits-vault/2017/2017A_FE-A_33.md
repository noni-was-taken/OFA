# 2017A_FE-A_33

In TCP/IP network, which of the following is a combination of information that identifies a TCP connection?

a) IP address, Session ID
b) IP address, Port number
c) MAC address, Session ID
d) Port number, Session ID
?
b) IP address, Port number
### Explanation

A TCP connection is uniquely identified by a **4-tuple**: {source IP, source port, destination IP, destination port}. The essential identifiers are **IP addresses** (to identify hosts) and **port numbers** (to identify the specific process/service on each host).

MAC addresses are Layer 2 identifiers used within a single network segment, and "Session ID" is not a TCP/IP concept.
