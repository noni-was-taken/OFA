# 2017S_FE-A_79
Which of the following is a method of making decisions based on the discount rate at which the net present value of the payback amount and the investment amount are equal?

a) Internal Rate of Return method
b) Net Present Value method
c) Payback Period method
d) Return on Investment method
?
a) Internal Rate of Return method
### Explanation

The **IRR (Internal Rate of Return)** is the discount rate at which the **NPV of all cash flows equals zero** — the present value of future returns exactly equals the investment.

- **NPV method**: calculates the net present value using a given discount rate
- **Payback Period**: time to recover the initial investment (ignores time value of money)
- **ROI**: ratio of net profit to investment cost
