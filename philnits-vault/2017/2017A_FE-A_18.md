# 2017A_FE-A_18

Which of the following is the backup method that requires the longest time to backup but the shortest time to restore?

a) Differential backup
b) Full backup
c) Incremental backup
d) Synthetic full backup
?
b) Full backup
### Explanation

| Method | Backup Time | Restore Time |
|--------|------------|--------------|
| **Full** | **Longest** (copies everything) | **Shortest** (single complete set) |
| Differential | Medium (changes since last full) | Fast (full + one differential) |
| Incremental | Shortest (changes since last backup) | Longest (full + all incrementals) |

**Full backup** takes the longest to perform but requires only one backup set to restore — making restoration the fastest.
