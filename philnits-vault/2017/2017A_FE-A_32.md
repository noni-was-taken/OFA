# 2017A_FE-A_32

Which of the following describes an IPv6 feature not existing in IPv4?

a) 128-bit addressing is introduced for the IP address space.
b) An IP address is represented as a pair of a network address and a subnet mask.
c) Private addresses are introduced to use IP addresses efficiently.
d) Subnet masks are introduced to use the address space efficiently.
?
a) 128-bit addressing is introduced for the IP address space.
### Explanation

The most fundamental difference between IPv4 and IPv6 is the address size:
- **IPv4** → 32-bit addresses
- **IPv6** → **128-bit addresses** (vastly larger address space)

The other features exist in IPv4:
- **B and D** → subnet masking and network/address representation exist in IPv4
- **C** → private addresses are an IPv4 concept (RFC 1918); IPv6 has a different approach
