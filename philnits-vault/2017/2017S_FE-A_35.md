# 2017S_FE-A_35
Which of the following is an Internet service that translates a domain name such as www.itpec.org into its corresponding IP address?

a) ARP
b) DNS
c) RARP
d) TCP
?
b) DNS
### Explanation

**DNS (Domain Name System)** resolves human-readable **domain names** into machine-readable **IP addresses**.

- **ARP**: maps IP addresses to MAC addresses on a local network
- **RARP**: maps MAC addresses back to IP addresses (legacy)
- **TCP**: a transport-layer protocol for reliable data transmission
