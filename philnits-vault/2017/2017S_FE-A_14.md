# 2017S_FE-A_14
Which of the following is an appropriate term to describe the phenomenon when files produced in the process of saving are scattered to different parts of the whole hard disk rather than in continuous clusters?

a) Contiguity
b) Fragmentation
c) Sector
d) Segmentation
?
b) Fragmentation
### Explanation

**Fragmentation** occurs when a file is stored in non-contiguous clusters scattered across a disk, typically as a result of repeated writes, deletions, and rewrites. This degrades read/write performance because the disk head must travel to multiple locations.
