# 2025A_FE-A_3
For an automaton with input and output symbols {0, 1} and a state transition diagram shown
below, which of the following is the output string for the input string 0011001110? Here, S1
represents the initial state, and the label x/y on each arc indicates that if x is an input, then y
is the corresponding output at the state transition. 

[2025A_FE-A_4.png]

a) 0001000110 
b) 0001001110
c) 0010001000 
d) 0011111110 
?
a) 0001000110 
### Explanation

For the automaton starting in state S1 with input string `0011001110`, trace each input symbol through the state transition diagram. At each step, the label `x/y` on the arc from the current state indicates: if input is `x`, output `y` and transition to the next state.

Following the state transitions step by step for each of the 10 input characters produces the output string **0001000110**.

### Answer: a) 0001000110
