# 2025A_FE-S_5
Among the four tree traversal techniques breadth-first, in-order, post-order, and pre-order which of the following is the appropriate combination to ensure that the last two nodes visited are the same in the tree shown below, when the left child is explored first?

[2025A_FE-S_5.png]

a) Breadth-First / Post-order
b) Breadth-First / Pre-order
c) In-order / Post-order
d) In-order / Pre-order
?
d) In-order / Pre-order
### Step 1: Understand what "last two nodes visited are the same" means

We need two different traversal methods that, when applied to the given tree, both end by visiting the same last two nodes in the same order.

### Step 2: Analyze traversal endings for the given tree

For the specific binary tree in this problem (exploring left child first):

- **Breadth-first (level-order)**: visits nodes level by level, left to right. The last nodes are the rightmost nodes at the deepest level.
- **In-order** (Left → Root → Right): the last node visited is the rightmost node in the tree, preceded by its parent.
- **Post-order** (Left → Right → Root): the last node visited is always the root of the tree.
- **Pre-order** (Root → Left → Right): the last node visited is the rightmost node reached by always going right from the deepest left subtree.

### Step 3: Identify matching pair

For this particular tree, tracing both **In-order** and **Pre-order** traversals results in the same final two nodes being visited. Both traversals end at the same rightmost path of the tree.

### Answer: d) In-order / Pre-order
