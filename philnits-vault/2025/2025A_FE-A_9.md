# 2025A_FE-A_9
When an HDD has the specifications shown in the table below, what is the average access time for the HDD to transfer 16 kB of data in milliseconds (ms)?

| Spec | Value |
|------|-------|
| Average seek time | 5 ms |
| Rotation Speed | 6,000 RPM |
| Transfer rate | 1 MB/s |
| Controller overhead | 0.1 ms |

a) 20.1
b) 25.725
c) 30.725
d) 74.1
?
b) 25.725
### Explanation

**Rotational latency** = half of one full rotation time
- 6,000 RPM → 1 rotation = $\frac{60}{6000} = 0.01$ s = 10 ms
- Average rotational latency = $\frac{10}{2} = 5$ ms

**Transfer time** for 16 kB:
- Transfer rate = 1 MB/s = 1,024 kB/s
- Transfer time = $\frac{16}{1024}$ s = $0.015625$ s = $15.625$ ms

**Total average access time:**
$5 \text{ (seek)} + 0.1 \text{ (controller)} + 5 \text{ (rotational latency)} + 15.625 \text{ (transfer)} = \mathbf{25.725}$ ms

### Answer: b) 25.725
